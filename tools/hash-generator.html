About Cryptographic Hashes

Cryptographic hash functions convert data of any size into a fixed-size hash value. They are one-way functions commonly used for data integrity verification and password storage.

Note: This tool uses the Web Crypto API which supports SHA-1, SHA-256, SHA-384, and SHA-512. MD5 is not available via this API.
